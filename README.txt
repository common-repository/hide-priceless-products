This plugin lets you to choose if you want to display the products without price or with a price set to 0

== Description ==

Have you ever wanted to hide products from your WooCommerce store that were either priced at zero or which price is simply uninformed? With this plugin, it is possible very easily! Hide products without price, and hide products with price set to 0.
Now you can even select categories of products that you want to keep visible. What else? Just ask!

== Installation ==

WooCommerce needs to be installed

1. Upload folder `hide-priceless-products` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

You find the settings under WooCommerce -> Settings -> Products

== Frequently Asked Questions ==

= Where do I find the settings? =
 
The settings can be found in the WooCommerce -> Settings -> Products general tab, at the bottom of the page. See also screenshot.
